# Analisis Strategi Algoritma 🚀

Repositori ini berisi kumpulan implementasi berbagai strategi algoritma menggunakan bahasa pemrograman Python. Proyek ini dikembangkan sebagai bagian dari tugas atau studi mengenai desain dan analisis algoritma.

## 📂 Struktur Proyek

Repositori ini dibagi menjadi beberapa kategori algoritma utama:

### 1. 📂 Brute Force

Algoritma yang melakukan pendekatan langsung (straightforward) untuk menyelesaikan masalah.

- `no-1.py`: String Matching (Pencocokan String)
- `no-2.py` s/d `no-4.py`: Implementasi Brute Force lainnya.

### 2. 📂 Divide and Conquer

Algoritma yang membagi masalah menjadi sub-masalah yang lebih kecil, menyelesaikannya secara rekursif, lalu menggabungkan hasilnya.

- [`binary-search.py`](./algoritma%20divide%20and%20conquer/binary-search.py): Pencarian biner pada array terurut.
- [`merge-sort.py`](./algoritma%20divide%20and%20conquer/merge-sort.py): Pengurutan data dengan metode penggabungan.
- [`maximum-subarray.py`](./algoritma%20divide%20and%20conquer/maximum-subarray.py): Mencari jumlah subarray terbesar dalam sebuah deret angka.

### 3. 📂 Greedy

Algoritma yang memilih langkah terbaik pada setiap tahap dengan harapan mencapai solusi optimal global.

- `no-1.py`: Masalah Penukaran Koin (Coin Change Problem).
- `no-2.py` s/d `no-6.py`: Implementasi algoritma Greedy lainnya.

## 🛠️ Cara Menjalankan Kode (Running the Code)

Pastikan Anda sudah menginstal [Python](https://www.python.org/downloads/) di sistem Anda sebelum memulai.

### Langkah-langkah:

1.  **Clone atau Download Repositori**
    Jika menggunakan Git:

    ```bash
    git clone <url-repository>
    cd <repository-name>
    ```

2.  **Buka Terminal atau Command Prompt**
    Arahkan ke direktori utama proyek.

3.  **Jalankan Script Python**
    Gunakan perintah `python` (atau `python3` di Linux/macOS) diikuti dengan path file yang ingin dijalankan.

    **Contoh menjalankan Binary Search:**

    ```bash
    python "algoritma divide and conquer/binary-search.py"
    ```

    **Contoh menjalankan Merge Sort:**

    ```bash
    python "algoritma divide and conquer/merge-sort.py"
    ```

    **Contoh menjalankan String Match (Brute Force):**

    ```bash
    python "algoritma brute force/no-1.py"
    ```
